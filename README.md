# Угадай число (l2-wb)

**Для локального запуска проекта необходимо выполнить следующие команды в терминале:**
```
git clone https://github.com/sazonov-dev/guess-the-number
```
```
cd guess-the-number
```
```
npm i
```
```
npm run dev
```
